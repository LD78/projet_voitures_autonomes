# projet_voitures_autonomes

1er version, avec app React de base...

# install

```
cd client > npm install
&& cd server > npm install
```


# dev

launch server

```
cd server && npm start
```
launch client

```
cd client && npm start
```
client: 
http://127.0.0.1:8080

# prod

```
npm start
```
server: 
http://127.0.0.1:3000

You have to launch mongod in your terminal before launch the application
